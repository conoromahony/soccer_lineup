About This Website

Hi there,

I created this website after consistently spending too much time painstakingly planning my team's substitution schedule each week in an attempt to ensure players got equal playing time. After doing this for a season, I thought there has to be a better way. I created this website to automate what I had been manually doing each week, and I thought I'd share it with others in case it can save them time as well. I hope it works for you. If you have suggestions on how to improve it, please let me know. I can't promise to incorporate every request, but if enough people are requesting a particular capability, I'll try to incorporate it.

Best regards,
Conor.
